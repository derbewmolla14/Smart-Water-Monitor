ስለ Smart Water Monitor

ሲስተሙ እንዴት ይሰራል?

ይህ ሲስተም የውሃ ማጠራቀሚያ ታንከሮችን መጠን በዘመናዊ መንገድ ለመቆጣጠር ታስቦ የተሰራ ነው። ሲስተሙ ሶስት ዋና ዋና ክፍሎች አሉት፦

  • Hardware (ሴንሰር): በታንከሩ አናት ላይ የሚገጠመው Ultrasonic Sensor የውሃውን ርቀት ይለካል።
  • Server (አንጎል): ESP32 መረጃውን በ Wi-Fi አማካኝነት ለ Node.js ሰርቨር ይልካል።
  • Web Dashboard (ማሳያ): ተጠቃሚው በየትኛውም ቦታ ሆኖ በድረ-ገጹ ላይ የውሃውን መጠን በፐርሰንት (%) ያያል።
